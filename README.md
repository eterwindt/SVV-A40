## Git Commands

**ALWAYS START WITH THIS**

### git pull 

Then, after making changes to files:

1) git add .
2) git commit -m "Describe what you changed here"
3) git push

If error:

4) git pull
    - resolve conflicts if necessary
    - (in "merge" menu): <kbd>ESC</kbd>+<kbd>:</kbd>+<kbd>wq</kbd>+<kbd>ENTER</kbd>
5) git push
